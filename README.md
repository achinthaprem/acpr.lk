# acpr.lk

This repo contains the source for **https://www.acpr.lk**, a simple static site hosted with GitHub Pages.

## Overview

- Static HTML, CSS, and optional JavaScript
- No build tools
- Deployed directly from the `main` branch via GitHub Pages
- Custom domain: `www.acpr.lk` (and optionally `acpr.lk`)

## Structure

Typical structure:

```text

├── index.html        # Main page

├── css/              # Stylesheets
├── js/               # JavaScript files
└── CNAME             # Custom domain config for GitHub Pages
